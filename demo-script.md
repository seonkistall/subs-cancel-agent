# 끊어줌 — 30초 데모 콘티 (Phase 0)

**목적:** "버튼 한 번에 구독이 *실제로* 취소되는 장면"을 보여줘 신뢰 + 선예약 유도.
**원칙:** 에이전트가 아직 없으니 **당신 테스트 계정에서 직접 취소하는 화면을 녹화**하고 "에이전트가 자동으로 한다"로 내레이션. 과장 금지(미래형 제품 데모임을 FAQ가 보강).
**세팅:** OBS/QuickTime 화면녹화. 지루한 로딩은 1.5~2배속. 두 버전 권장 — 16:9(랜딩), 9:16(쇼츠/릴스).

| 시간 | 화면(샷) | 내레이션 / 자막 |
|---|---|---|
| 0–4s | 카드 명세서 또는 결제 알림이 우르르 | "안 쓰는데 매달 빠지는 구독, 끊는 건 왜 이렇게 귀찮죠?" |
| 4–9s | 끊어줌 화면: 구독 리스트 + 한 항목 옆 '끊기' 버튼 클릭 | "끊고 싶은 구독 옆 '끊기' 한 번만." |
| 9–20s | (실제 녹화·배속) 해지 페이지 진입 → 리텐션 할인 팝업 무시 → 최종 '구독 취소' 클릭까지 | "에이전트가 당신 대신 들어가, 해지 페이지를 찾고, 붙잡는 유혹을 무시하고, 끝까지 취소합니다." |
| 20–26s | '취소 완료' 화면 + 벤더가 보낸 해지 확인 메일 | "비밀번호는 안 받아요. 당신 기기에서, 당신 세션으로. 안 되면 전액 환불." |
| 26–30s | 엔드카드: 로고 + "지금 선예약 ₩9,900 · 실패 시 환불" | "끊어줌. 지금 선예약하세요." |

## 촬영 팁
- 1번 벤더는 "웹에서 해지되고, 리텐션 팝업이 있는" 걸로 골라야 데모에 드라마가 산다. 예: 쿠팡 와우/티빙/왓챠, 또는 해외 SaaS.
- 마우스 동선을 또렷하게, 클릭은 하이라이트(클릭 강조 툴).
- 무음 자막만으로도 OK(쇼츠는 무음 시청이 많음).
- 진짜 해지까지 가기 싫으면: 해지 직전 '최종 확인' 버튼 앞에서 멈추고 "여기까지 자동" 컷 처리도 가능.
