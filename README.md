# 끊어줌 — Phase 0 선예약 검증 랜딩

목표: **"한국인이 구독 취소 대행에 ₩9,900을 내는가"** 를 코드 풀빌드 없이 측정한다.
- **하드 신호** = Toss로 ₩9,900 결제한 사람 수
- **소프트 신호 + 벤더 수요** = 대기자 폼 제출(어떤 구독을 끊고 싶은지 = 벤더 #1 힌트)

설계 근거: `~/.gstack/projects/subs-cancel-agent/ktykj-concept-design-20260628-112119.md`

## 배포 전, 딱 4곳만 수정 (`index.html`의 `★ EDIT` 주석)

1. **Toss 결제 링크** — `data-toss="https://toss.me/your-link"` 2곳, 또는 `<script>`의 `ONE_LINK` 한 줄에 본인 링크.
   - 가장 빠른 방법: **toss.me** 개인 송금 링크(금액 고정 ₩9,900)로 시작.
   - 제대로 하려면: Toss Payments 가입 → 결제(payment) 링크 생성 → 그 URL 사용. (사업자/정산·세금계산서까지 깔끔)
2. **Formspree ID** — `<form action="https://formspree.io/f/XXXXXXX">` → [formspree.io](https://formspree.io) 무료 가입 후 받은 엔드포인트. (백엔드 0줄로 폼 수집)
3. **데모 영상** — `<div class="demo-frame">` 안을 본인 30초 화면녹화로 교체.
   - 만드는 법: **본인 테스트 계정**에서 구독 1개를 실제로 취소하는 화면을 녹화 → "우리 에이전트가 이걸 자동으로" 자막. 유튜브 업로드 후 `<iframe>` 임베드하거나 gif.
4. **브랜드명** — "끊어줌"이 마음에 안 들면 전역 치환.

| 단계 | 표본 | 통과 기준 |
|---|---|---|
| 스모크 | 보여준 10명 | ₩9,900 결제 **≥ 2~3명** → 확대 / 0명 → 경고 |
